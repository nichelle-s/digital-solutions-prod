# Digital Solutions - Production Environment Smulation

## Overview
This project demonstrates the ability to manage and operate a multi-instance cloud environment within AWS. The initial infrastructure was established using AI-generated Terraform and bootstrap scripts to create a realistic "inherited" corporate stack for a mock company, "Digital Solutions."

## Goal
The objective is to validate technical competency in a professional AWS environment by performing manual operational verification, mapping infrastructure dependencies, and conducting troubleshooting and health checks across a live production fleet.

## Scope
* **Environment Mapping**: Identifying and documenting compute resources, network boundaries, and security constraints.
* **Flow Analysis**: Tracing the paths of administrative access, public traffic, and automated monitoring alerts.
* **Operational Inspection**: Validating the health of services and persistence layers within the AWS environment.

## Technology Stack
* **Cloud Infrastructure**: AWS
